# Vue 3 + Vite
[Vue3](https://v3.cn.vuejs.org/guide/introduction.html)
[Vite](https://cn.vitejs.dev/)

## 使用Vite构建项目
npm:
```
# npm 6.x
$ npm init vite@latest <project-name> --template vue

# npm 7+，需要加上额外的双短横线
$ npm init vite@latest <project-name> -- --template vue

$ cd <project-name>
$ npm install
$ npm run dev
```
yarn:
```
$ yarn create vite <project-name> --template vue
$ cd <project-name>
$ yarn
$ yarn dev
```
